# Plan: Add Nova Chat Avatar to All Pages

## Goal
Make the Nova chat avatar appear on all pages in the CHEA app by adding the `NovaChat` component to the dashboard layout.

## Analysis
- **NovaChat** component: `project/src/components/ui/NovaChat.tsx` - A floating chat widget with an avatar that can be dragged and clicked to open a chat panel
- **DashboardLayout**: `project/src/components/layout/DashboardLayout.tsx` - The layout that wraps all protected/dashboard routes
- **Current State**: The NovaChat component exists but is NOT being rendered anywhere in the app

## Implementation Steps

1. **Import NovaChat in DashboardLayout**
   - Add import statement for `NovaChat` from `@/components/ui/NovaChat`

2. **Add NovaChat Component to DashboardLayout**
   - Place `<NovaChat />` inside the DashboardLayout component, preferably near the end of the layout (after the mobile sidebar overlay)
   - The component is already designed as a fixed-position element, so it will float above all content

3. **Verify Component Export**
   - Ensure `NovaChat` is exported as a named export (it's already exported as `export function NovaChat`)

## Expected Result
- The Nova chat avatar will appear as a floating button on the bottom-right of all dashboard pages
- Users can drag it to reposition
- Clicking opens the chat panel
- Works with both light and dark themes
